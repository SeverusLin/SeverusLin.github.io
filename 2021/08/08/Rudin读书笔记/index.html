Chapter 1 The Real and Complex Number Systems

Introduction

1.1 Example

Equation \(p^2=2\) is not satisfied by any rational \(p\) .

Let \(A\) be the set of all positive rationals \(p\), \(s.t.\) \(p^2<2\).

Let \(B\) be the set of all positive rationals \(p\), \(s.t.\) \(p^2>2\).

And \(A\) has no largest number while \(B\) has no smallest number.

Let

\[ \begin{equation}q=p-\dfrac{p^2-2}{p+2}=\dfrac{2p+2}{p+2}\label{intro1}\end{equation} \] then,

\[ \begin{equation}q^2-2=\dfrac{2(p^2-2)}{(p+2)^2}\label{intro2}\end{equation} \] For each \(p\) in \(A\), \(q>p\) and \(q\) is in \(A\) .

For each \(p\) in \(B\), \(q<p\) and \(q\) is in \(B\) .

1.2 Remark

The rational number system has certain gaps and the real number system will fill them.

1.3 Definition

  • \(x\in A\), \(x\) is a member of \(A\).
  • \(x\notin A\), \(x\) is not a member of \(A\).
  • \(\varnothing\), empty set
  • \(A\subset B\)

1.4 Definition

\(\mathbb{Q}\), the rational number.

Ordered Sets

1.5 Definition

Let \(S\) be a set. An order on \(S\) is a relation, denoted by \(<\), with the following two properties:

  1. If \(x\in S\) and \(y\in S\) then one and only one of the statements \[ \begin{equation*} x<y,\quad x=y, \quad y<x \end{equation*} \] is true.

  2. If \(x,y,z\in S\), if \(x<y\) and \(y<z\), then \(x<z\).

Notation: \(<, >, =, \leqslant, \geqslant\).

Negation of \(x<y\) is \(x\geqslant y\).

1.6 Definition

An ordered set is a set is a set \(S\) in which an order is defined.

1.7 Definition

Suppose \(S\) is an ordered set , and \(E\subset S\). If \(\exists \beta\in S, s.t. x\leqslant \beta\) for every \(x\in E\), we say that \(E\) is bounded above, and call \(\beta\) an upper bound of \(E\).

Lower bounds can be defined in the same way.

1.8 Definition

Suppose \(S\) is an ordered set, \(E\subset S\), and \(E\) is bounded above. Suppose \(\exists \alpha\in S\) with the following properties:

​ (a) \(\alpha\) is an upper bound of \(E\).

​ (b) If \(\gamma<\alpha\) then \(\gamma\) is not an upper bound of \(E\).

​ (b') (equivalent form) \(\forall \beta<\alpha,\exists\gamma\in E,s.t. \beta<\gamma\).

Then \(\alpha\) is called the least upper bound of \(E\), or the supremum of \(E\).

We denote, \[ \begin{equation*} \alpha = \sup E \end{equation*} \] The greatest lower bound, or infimum of \(E\) (which is bounded below) is defined in the same manner. \[ \begin{equation*} \alpha = \inf E \end{equation*} \]

1.9 Examples

(a) Sets \(A\) and \(B\) in 1.1

\(A\) has no supremum in \(\mathbb{Q}\).

\(B\) has no infimum in \(\mathbb{Q}\).

(b) \(\alpha=\sup E\), but \(\alpha\) may or may not be a member of \(E\).

(c) \(E=\{ 1/n |n\in{ \mathbb{Z}_{+} } \}\), \(\inf E=0\), but \(0\notin E\).

1.10 Definition

An ordered set \(S\) is said to have the least-upper-bound property if the following is true:

If \(E\subset S\), \(E\) is not empty, and \(E\) is bounded above, then \(\sup E\) exists in \(S\).

\(\mathbb{Q}\) does not have the least-upper-bound property.

1.11 Theorem

Suppose \(S\) is an ordered set with the least-upper-bounded property, \(B\subset S\), \(B\) is not empty, and \(B\) is bounded below. Let \(L\) be the set of all lower bounds of \(B\).Then \[ \begin{equation*} \alpha = \sup L \end{equation*} \] exists in \(S\), and \(\alpha = \inf B\).

In particular, \(\inf B\) exists in \(S\).

Proof, For \(B\) is bounded below,\(L\neq\varnothing\). \(\forall x\in L, y\in B\) satisfy that \(x\leqslant y\). And each \(y\in B\) is a upper bounder of \(L\). Thus, \(L\) has a supremum \(\alpha = \sup L\) in \(S\).

\(\forall\gamma < \alpha, \gamma\leqslant y(\forall y\in B)\), thus \(\alpha \leqslant y(\forall y\in B)\) and \(\alpha\) is a lower bound of \(B, i.e. \alpha \in L\). Since \(\alpha\) is an upper bound of \(L\), \(\forall \beta>\alpha\), \(\beta\notin L\)(Set of lower bounds). Hence, \(\alpha\) is the infimum of \(B\).

Field

1.12 Definition

A field is a set \(F\) with two operations, called addition and multiplication, which satisfy the following so-called "field axioms"(A),(M) and (D):

(A) Axioms of addition

(A1) If \(x\in F\) and \(y\in F\), then their sum \(x+y\in F\).

(A2) Addition is commutative: \(\forall x,y\in F, x+y=y+x\).

(A3) Addition is associative: \(\forall x,y,z\in F, (x+y)+z=x+(y+z)\).

(A4) \(F\) contains an element \(0, s.t. \forall x\in F\quad 0+x=x\).

(A5) \(\forall x \in F\) corresponds an element \(-x\in F, s.t.\quad x+(-x)=0\).

(M) Axioms of multiplication

(M1) If \(x\in F\) and \(y \in F\), then their product \(xy\in F\).

(M2) Multiplication is commutative: \(\forall x,y \in F,xy =yx\).

(M3) Multiplication is associative: \(\forall x,y,z\in F, (xy)z=x(yz)\).

(M4) \(F\) contains an element \(1\neq 0, s.t.\forall x \in F,\quad 1x=x\).

(M5) If \(x\in F\) and \(x\neq 0\) then \(\exists 1/x\in F, s.t. \quad x(1/x)=1\).

(D) The distribution law \[ \begin{equation*} x(y+z)=xy+xz \end{equation*} \] holds for all \(x,y\in F\).

1.13 Remarks

(a) Some notations: \(x-y,\dfrac{x}{y},x+y+z,xyz,x^2,x^3,2x,3x,\ldots\)

(b) \(\mathbb{Q}\) is a field.

(c) Properties from field axioms are familiar in rational numbers, real numbers and complex numbers.

1.14 Proposition

The axioms for addition imply the following statements.

(a) If \(x+y=x+z\) then \(y=z\). (Cancellation law)

(b) If \(x+y=x\) then \(y=0\).

(c) If \(x+y=0\) then \(y=-x\).

(d) \(-(-x)=x\).

Proof, Omitted.

1.15 Proposition

The axioms for multiplication imply the following statements.

(a) If \(x\neq 0\) and \(xy=xz\) then \(y=z\).

(b) If \(x\neq 0\) and \(xy=x\) then \(y=1\).

(c) If \(x\neq 0\) and \(xy=1\) then \(y=1/x\).

(d) If \(x\neq 0\) then \(1/(1/x)=x\).

Proof, Omitted.

1.16 Proposition

The field axioms imply the following statements, for any $ x,y,zF$.

(a) \(0x=0\).

(b) If \(x\neq 0\) and \(y\neq 0\) then \(xy\neq 0\).

(c) \((-x)y=-(xy)=x(-y)\).

(d) \((-x)(-y)=xy\).

Proof, Omitted.

1.17 Definition

An ordered field is a field \(F\) which is also an ordered set, such that

(a) \(x+y<x+z\), if \(x,y,z\in F\) and \(y<z\).

(b) \(xy>0\), if \(x\in F, y\in F,x>0,y>0\).

If \(x>0(x<0)\), we call \(x\) positive(negative).1

1.18 Proposition

The following statements are true in every ordered field.

(a) If \(x>0\) then \(-x<0\) and vice versa.

(b) If \(x>0\) and \(y< z\) then $ xy< xz $.

(c) If \(x<0\) and \(y< z\) then $ xy>xz $.

(d) If \(x\neq0\) then \(x^2>0\). In particular, \(1>0\).

(e) If \(0<x<y\) then \(0<1/y<1/x\).

Proof, Omitted.

The Real Field

1.19 Theorem

There exists an ordered field \(\mathbb{R}\) which has the least-upper-bound property. Moreover, \(\mathbb{R}\) contains \(\mathbb{Q}\) as a subfield ($ $).

Proof, See the Appendix to Chap.1.

1.20 Theorem

(a) \(\forall x,y\in\mathbb{R},x>0, \exists n\in\mathbb{N},s.t. nx>y\).(Archimedean property)

(b) \(\forall x,y\in\mathbb{R}, x<y, \exists p\in\mathbb{Q}, s.t. x<p<y\).(\(\mathbb{Q}\) is dense in \(\mathbb{R}\))

Proof, (a) Let \(A=\left\{nx|x\in\mathbb{R},n\in\mathbb{N}\right\}\). If (a) is false, then \(y\) would be an upper bound of \(A\). Thus, suppose \(\alpha=\sup A\). Since \(\alpha-x<\alpha\), \(\exists n_0\in\mathbb{N}, s.t. \alpha-x<n_0x<\alpha\). Thus, \(\alpha<(n_0+1)x\), contradicted to that \(A\) is upper-bounded.

(b) From (a) we obtain a positive integer \(n\) such that, \[ \begin{equation*} n(y-x)>1 \end{equation*} \] Apply (a) again, positive integers \(m_1\) and \(m_2\) are obtained such that \(m_1>nx\) and \(m_2>-nx\). Then, \[ \begin{equation*} -m_2<nx<m_1 \end{equation*} \] Hence, \(\exists m\in \mathbb{N},-m_2\leqslant m\leqslant m_1\), such that, \[ \begin{equation*} m-1\leqslant nx < m \end{equation*} \] Then, combine these inequalities, \[ \begin{equation*} nx<m\leqslant 1+nx<ny. \end{equation*} \] Since \(n>0\), \[ \begin{equation*} x<\dfrac{m}{n}<y \end{equation*} \]

1.21 Theorem

\(\forall x\in\mathbb{R}_+,\forall n\in\mathbb{Z}_+,\exists ! y\in \mathbb{R}_+,s.t. y^n=x\).

This number \(y\) is written \(\sqrt[n]{x}\) or \(x^{1/n}\).

Sketch of Proof, Let \(E=\left\{t|t^n<x\right\}\), and \(y=\sup E\). And find contradictions when \(y^n\neq x\).

Choose \(h<\dfrac{x-y^n}{n(y+1)^{n-1}}\),

Try to prove that when \(y^n<x\), \(y+h\in E\) but \(y<y+h\).

Choose \(k=\dfrac{x-y^n}{ny^{n-1}}\),

Try to prove that when \(y^n>x\), \(y-k\) is an upper bound of \(E\) but \(y-k<y\).

Corollary

\(\forall a,b\in\mathbb{R}_+, n\in\mathbb{N}_+, (ab)^{1/n}=a^{1/n}b^{1/n}\).

Proof, Hint: Let\(\alpha = a^{1/n},\beta=b^{1/n}\), and use Thm. 1.21.

1.22 Decimal

Suppose \(x\in\mathbb{R}_+\). Then choose integer \(n_0\) such that \(n_0\leqslant x\). Similarly choose integers \(n_1,n_2,\ldots,n_k\) such that \[ \begin{equation*} n_0+\dfrac{n_1}{10}+\ldots+\dfrac{n_k}{10^k}\leqslant x \end{equation*} \] Let \(E\) be the set of these numbers \[ \begin{equation*} n_0+\dfrac{n_1}{10}+\ldots+\dfrac{n_k}{10^k}\leqslant x\qquad(k=0,1,\ldots) \end{equation*} \] Then \(x=\sup E\). The decimal expansion of \(x\) is \[ \begin{equation*} n_0.n_1n_2n_3\ldots \end{equation*} \]

The Extended Real Number System

1.23 Definition

The extended real number system \(\overline{\mathbb{R}}=\mathbb{R}\cup{\pm\infty}\) .

And \(\forall x\in\mathbb{R}, -\infty<x<+\infty\).

\(\overline{\mathbb{R}}\) is not a field.

\(+\infty\) is an upper bound of \(\overline{\mathbb{R}}\).

(a) \(\forall x\in\mathbb{R}\), \[ \begin{equation*} x+\infty=+\infty,\qquad x-\infty=-\infty,\qquad \dfrac{x}{+\infty}=\dfrac{x}{-\infty}=0. \end{equation*} \] (b) \(\forall x\in\mathbb{R}_+, x\cdot(+\infty)=+\infty, x\cdot(-\infty)=-\infty\).

(c) \(\forall x\in\mathbb{R}_-, x\cdot(+\infty)=-\infty, x\cdot(-\infty)=+\infty\).

The Complex Field

1.24 Definition

A Complex number is an ordered pair \((a,b)\) of real numbers.

Let \(x=(a,b),y=(c,d)\), and \(x=y\) if and only if \(a=c,b=d\).

Addition is defined as \(x+y=(a+c,b+d)\).

Multiplication is defined as \(xy=(ac-bd,ad+bc)\).

1.25 Theorem

These definitions of addition and multiplication turn the set of all complex numbers into a field, with \((0,0)\) and \((1,0)\) in the role of \(0\) and \(1\).

Proof, Verify the field axioms.

1.26 Theorem

\(\forall a,b\in\mathbb{R},(a,0)+(b,0)=(a+b,0),\quad (a,0)(b,0)=(ab,0)\).

Proof, The proof is trivial.

1.27 Definition

\(i=(0,1)\).

1.28 Theorem

\(i^2=-1\).

Proof, The proof is trivial.

1.29 Theorem

\(\forall a,b\in\mathbb{R},(a,b)=a+bi\).

Proof, The proof is trivial.

1.30 Definition

\(z=a+bi\in\mathbb{C}\)

The conjugate of \(z\) is \(\bar{z}=a-bi\).

The real part of \(z\) is \(a=Re(z)\).

The imaginary part of \(z\) is \(b=Im(z)\).

1.31 Theorem

\(\forall z,w\in\mathbb{C}\),

(a) \(\overline{z+w}=\overline{z}+\overline{w}\).

(b) \(\overline{zw}=\bar{z}\bar{w}\)

(c) \(z+\bar{z}=2Re(z),z-\bar{z}=2Im(z)\).

(d) \(z\bar{z}\) is real and non-negative.

Proof, Omitted.

1.32 Definition

\(z\in \mathbb{C}\), the absolute value of \(z\) is \(|z|=(z\bar{z})^{1/2}\).

1.33 Theorem

\(z,w\in\mathbb{C}\),

(a) \(|z|>0\) if \(z=0\), \(|z|=0\) if \(z=0\).

(b) \(|\bar{z}|=|z|\).

(c) \(|zw|=|z||w|\).

(d) \(|Re(z)|\leqslant|z|\).

(e) \(|z+w|\leqslant|z|+|w|\).

Proof, Omitted.

1.34 Notation

\(x_1,\ldots,x_n\in\mathbb{C}\),

\(\displaystyle\sum_{j=1}^{n}x_j=x_1+\ldots+x_n\).

1.35 Theorem

(Schwarz inequality)

If \(a_1,\ldots,a_n\) and \(b_1,\ldots,b_n\) are complex numbers, then \[ \begin{equation*} \left|\sum_{j=1}^{n}a_j\bar{b_j}\right|^2\leqslant \sum_{j=1}^{n}\left|a_j\right|^2\sum_{j=1}^{n}\left|b_j\right|^2. \end{equation*} \]

Proof, Hint: use Lagrange's identity.

Euclidean Spaces

1.36 Definition

For each positive integer \(k\), let \(\mathbb{R}^k\) be the set of all ordered \(k\)-tuples, \(\mathbf{x}=(x_1,\ldots,x_k)\), where \(x_1,\ldots,x_k\in \mathbb{R}\) is called coordinate of \(\mathbf{x}\). The elements in \(\mathbb{R}^k\) are called points. If \(\mathbf{y}=(y_1,\ldots,y_k)\) and if \(\alpha\) is a real number, put \[ \begin{align*} \mathbf{x}+\mathbf{y}&=(x_1+y_1,\ldots,x_k+y_k),\\ \alpha\mathbf{x}&=(\alpha x_1,\ldots,\alpha x_k) \end{align*} \] so that \(\mathbf{x}+\mathbf{y}\in\mathbb{R}^k\) and \(\alpha\mathbf{x}\in\mathbb{R}^k\). \(\mathbb{R}^k\) is a vector space over the real field. The zero element of \(\mathbb{R}^k\) is \(\mathbf{0}=(0,\ldots,0)\).

Inner product(scalar product), \(\mathbf{x}\cdot\mathbf{y}=\displaystyle\sum_{i=1}^k x_i y_i\).

The norm of \(\mathbf{x}\), \(|\mathbf{x}|=(\mathbf{x}\cdot\mathbf{x})^{1/2}=\left(\displaystyle\sum_{i=1}^k x_i^2\right)^{1/2}\).

The structure defined is called Euclidean \(k\)-space.

1.37 Theorem

\(\mathbf{x},\mathbf{y},\mathbf{z}\in\mathbb{R}^k\), and \(\alpha\) is real. Then,

(a) \(|\mathbf{x}|\geqslant 0\).

(b) \(|\mathbb{x}=0|=0 \Leftrightarrow \mathbf{x}=\mathbf{0}\).

(c) \(|\alpha\mathbf{x}|=|\alpha||\mathbf{x}|\).

(d) \(|\mathbf{x}\cdot\mathbf{y}|\leqslant|\mathbf{x}||\mathbf{y}|\).

(e) \(|\mathbf{x}+\mathbf{y}|\leqslant|\mathbf{x}|+|\mathbf{y}|\).

(f) \(|\mathbf{x}-\mathbf{z}|\leqslant|\mathbf{x}-\mathbf{y}|+|\mathbf{y}-\mathbf{z}|\).

Proof, Omitted.

1.38 Remarks

\(\mathbb{R}^k\) can be regarded as a metric space.

Appendix

Theorem 1.19 will be proved in this appendix by constructing \(\mathbb{R}\) from \(\mathbb{Q}\).

We denote rational numbers with English letters \(p,q,r,\ldots\),while Greek letters \(\alpha,\beta,\gamma,\ldots\), are used to denote the cuts.

This method used here were invented by Dedekind.

(Cantor constructed \(\mathbb{R}\) from \(\mathbb{Q}\) by means of Cauchy sequences.)

Step 1

The members of \(\mathbb{R}\) will be certain subsets of \(\mathbb{Q}\), called cuts.

Definition of a cut: any set \(\alpha\subset\mathbb{Q}\) with the following properties.

  1. \(\alpha\) is not empty, and \(\alpha\neq\mathbb{Q}\).
  2. If \(p\in\alpha,q\in\mathbb{Q}\), and \(q<p\), then \(q\in\alpha\).
  3. If \(p\in\alpha\), then \(p<r\) for some \(r\in\alpha\).

Some facts:

  • \(\alpha\) has no largest member.
  • If \(p\in\alpha, q\notin\alpha\) then \(p<q\).
  • If \(r\in\alpha, r<s\) then \(s\notin\alpha\).

Step 2

Define ''\(\alpha<\beta\)'' to mean: \(\alpha\subsetneq\beta\).

This meets the requirement of Def. 1.5.(Why?)

And only one in three relations can hold for any pair \(\alpha,\beta\).

\(\mathbb{R}\) is an ordered set.

Step 3

The ordered set \(\mathbb{R}\) has the least-upper-bound property.

Let \(A\subset\mathbb{R},A\neq\varnothing\), and assume that \(\beta\in\mathbb{R}\) is an upper bound of \(A\).

Define \(\gamma\) to be the union of all \(\alpha\in A\). ( If and only if $ p$ for some \(\alpha\in A\))

  1. Verify that \(\gamma\) meets the properties in Step 1, then \(\gamma\in\mathbb{R}\).
  2. Verify that \(\gamma=\sup A\) (Def. 1.8).

Step 4

Axioms of Addition

If \(\alpha,\beta\in\mathbb{R}\), we define \(\alpha+\beta\) to be the set of all sums \(r+s\), where \(r\in\alpha,s\in\beta\).

\(0^*\) is defined to be the set of all negative rational numbers.(\(0^*\) is a cut.)

Please Verify that the axioms of addition (see Def. 1.12) hold in \(\mathbb{R}\), with \(0^*\) playing the role of \(0\).

Step 5

(a) in Def. 1.17 can be proved.

Prop. 1.14 is valid in \(\mathbb{R}\).

Step 6

Multiplication

Define \(\mathbb{R}_+\) to be the set of all \(\alpha\in\mathbb{R}\) with \(\alpha>0^*\).

\(\alpha,\beta\in\mathbb{R}_+\), \(\alpha\beta\) is define to be the set of all \(p\) such that \(p\leqslant rs\) for some choice of \(r\in\alpha,s\in\beta,r>0,s>0\).

\(1^*\) is defined to be the set of all \(q<1\).

Then the axioms (M) and (D) in Def. 1.12 hold in \(\mathbb{R}_+\) with \(1^*\) in the role of \(1\).

(b) in Def. 1.17 holds now.

Step 7

Complete the definition of multiplication

Set \(\alpha 0^*=0^*\alpha=0^*\).

Set \[ \begin{equation*} \alpha\beta=\left\{ \begin{aligned} (-\alpha)(-\beta),\qquad\alpha<0^*,\beta<0^*\\ -[(-\alpha)\beta],\qquad\alpha<0^*,\beta>0^*\\ -[\alpha\cdot(-\beta)],\qquad\alpha>0^*,\beta<0^* \end{aligned} \right. \end{equation*} \] By the identity \(-(-\gamma)=\gamma\), we can prove that the axioms (M) hold in \(\mathbb{R}\).

Try to prove distribution law (D) in \(\mathbb{R}\). (Break it into cases.)

Till now, \(\mathbb{R}\) is an ordered field with the least-upper-bound property.

Step 8

\(r^*\) is associated with \(r\in\mathbb{Q}\), which is defined to be the set of all \(p\in\mathbb{Q}\) such that \(p<r\).

\(r^*\) is a cut and the following relations are satisfied,

(a) \(r^*+s^*=(r+s)^*\).

(b) \(r^*s^*=(rs)^*\).

(c) \(r*<s*\) if and only if \(r<s\).

Here we prove (a).

Choose \(p\in r^*+s^*\). Then \(p=u+v\), where \(u<r,v<s\). Hence \(p<r+s\), which says that \(p\in(r+s)^*\).

Conversely, suppose \(p\in(r+s)^*\). Then \(p<r+s\). Choose \(t\) so that \(2t=r+s-p\), put \[ \begin{equation*} r^{\prime}=r-t, s^{\prime}=s-t. \end{equation*} \] Then \(r^{\prime}\in r^*, s^{\prime}\in s^*\), and \(p=r^\prime+s^\prime\), so that \(p\in r^*+s*\).

Try to prove (b),(c).

Step 9

The ordered field \(\mathbb{Q}\) is isomorphic to the order field \(\mathbb{Q}^*\) we establish with the method in Step 8.

This identification of \(\mathbb{Q}\) with \(\mathbb{Q}^*\) allows us to regard \(\mathbb{Q}\) as a subfield of \(\mathbb{R}\).

The fact below holds,

Any two ordered field with the least-upper-bound property are isomorphic

Exercises

Q5. Let \(A\) be a nonempty set of real numbers which is bounded below. Let \(-A\) be the set of all numbers \(-x\), where \(x\in A\). Prove that \[ \begin{equation*} \inf A=-\sup (-A). \end{equation*} \] Q6. Fix \(b>1\).

(a) If \(m,n,p,q\) are integers, \(n>0,q>0\), and \(r=m/n=p/q\), prove that \[ \begin{equation*} (b^m)^{1/n}=(b^p)^{1/q}. \end{equation*} \] Hence it makes sense to define \(b^r=(b^m)^{1/n}\).

(b) Prove that \(b^{r+s}=b^rb^s\) if \(r\) and \(s\) are rational.

(c) If \(x\) is real, define \(B(x)\) to be the set of all numbers \(b^t\), where \(t\) is rational and \(t\leqslant x\). Prove that \[ \begin{equation*} b^r=\sup B(r) \end{equation*} \] when \(r\) is rational. Hence it makes sense to define \[ \begin{equation*} b^x=\sup B(x) \end{equation*} \] for every real \(x\).

(d) Prove that \(b^{x+y}=b^xb^y\) for all real \(x\) and \(y\).

Q7. Fix \(b>1, y>0\), and prove that there is a unique real \(x\) such that \(b^x=y\), by completing the following outline. (This \(x\) is called the logarithm of \(y\) to the base \(b\).)

(a) For any positive integers \(n\), \(b^n-1\geqslant n(b-1)\).

(b) Hence \(b-1\geqslant n(b^{1/n}-1)\).

(c) If \(t>1\) and \(n>(b-1)/(t-1)\), then \(b^{1/n}<t\).

(d) If \(w\) is such that \(b^w<y\), then \(b^{w+(1/n)}<y\) for sufficiently large \(n\); to see this, apply part (c) with \(t=y\cdot b^{-w}\).

(e) If \(b^w>y\), then \(b^{w-(1/n)}>y\) for sufficiently large \(n\).

(f) Let \(A\) be the set of all \(w\) such that \(b^w<y\), and show that \(x=\sup A\) satisfies \(b^x=y\).

(g) Prove that this \(x\) is unique.

Q20. With reference to the Appendix, suppose that property 3. were omitted from the definition of a cut. Keep the same definitions of order and addition. Show that the resulting ordered set has the least-upper-bound property, that addition satisfies axioms (A1) to (A4) (with a slightly different zero-element!) but that (A5) fails.
