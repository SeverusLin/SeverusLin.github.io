Ordered Sets

1.5 Definition

Let $S$ be a set. An order on $S$ is a relation, denoted by $<$, with the following two properties:

  1. If $x\in S$ and $y\in S$ then one and only one of the statements

    is true.

  2. If $x,y,z\in S$, if $x<y$ and $y<z$, then $x<z$.

Notation: $<, >, =, \leqslant, \geqslant$.

Negation of $x<y$ is $x\geqslant y$.

1.6 Definition

An ordered set is a set is a set $S$ in which an order is defined.

1.7 Definition

Suppose $S$ is an ordered set , and $E\subset S$. If $\exists \beta\in S, s.t. x\leqslant \beta$ for every $x\in E$, we say that $E$ is bounded above, and call $\beta$ an upper bound of $E$.

Lower bounds can be defined in the same way.

1.8 Definition

Suppose $S$ is an ordered set, $E\subset S$, and $E$ is bounded above. Suppose $\exists \alpha\in S$ with the following properties:

​ (a) $\alpha$ is an upper bound of $E$.

​ (b) If $\gamma<\alpha$ then $\gamma$ is not an upper bound of $E$.

​ (b’) (equivalent form) $\forall \beta<\alpha,\exists\gamma\in E,s.t. \beta<\gamma$.

Then $\alpha$ is called the least upper bound of $E$, or the supremum of $E$.

We denote,

The greatest lower bound, or infimum of $E$ (which is bounded below) is defined in the same manner.

1.10 Definition

An ordered set $S$ is said to have the least-upper-bound property if the following is true:

If $E\subset S$, $E$ is not empty, and $E$ is bounded above, then $\sup E$ exists in $S$.

$\mathbb{Q}$ does not have the least-upper-bound property.

Field

1.12 Definition

A field is a set $F$ with two operations, called addition and multiplication, which satisfy the following so-called “field axioms”(A),(M) and (D):

(A) Axioms of addition

(A1) If $x\in F$ and $y\in F$, then their sum $x+y\in F$.

(A2) Addition is commutative: $\forall x,y\in F, x+y=y+x$.

(A3) Addition is associative: $\forall x,y,z\in F, (x+y)+z=x+(y+z)$.

(A4) $F$ contains an element $0, s.t. \forall x\in F\quad 0+x=x$.

(A5) $\forall x \in F$ corresponds an element $-x\in F, s.t.\quad x+(-x)=0$.

(M) Axioms of multiplication

(M1) If $x\in F$ and $y \in F$, then their product $xy\in F$.

(M2) Multiplication is commutative: $\forall x,y \in F,xy =yx$.

(M3) Multiplication is associative: $\forall x,y,z\in F, (xy)z=x(yz)$.

(M4) $F$ contains an element $1\neq 0, s.t.\forall x \in F,\quad 1x=x$.

(M5) If $x\in F$ and $x\neq 0$ then $\exists 1/x\in F, s.t. \quad x(1/x)=1$.

(D) The distribution law

holds for all $x,y\in F$.

1.13 Remarks

(a) Some notations: $x-y,\dfrac{x}{y},x+y+z,xyz,x^2,x^3,2x,3x,\ldots$

(b) $\mathbb{Q}$ is a field.

(c) Properties from field axioms are familiar in rational numbers, real numbers and complex numbers.

1.14 Proposition

The axioms for addition imply the following statements.

(a) If $x+y=x+z$ then $y=z$. (Cancellation law)

(b) If $x+y=x$ then $y=0$.

(c) If $x+y=0$ then $y=-x$.

(d) $-(-x)=x$.

Proof, Omitted.

1.15 Proposition

The axioms for multiplication imply the following statements.

(a) If $x\neq 0$ and $xy=xz$ then $y=z$.

(b) If $x\neq 0$ and $xy=x$ then $y=1$.

(c) If $x\neq 0$ and $xy=1$ then $y=1/x$.

(d) If $x\neq 0$ then $1/(1/x)=x$.

Proof, Omitted.

1.16 Proposition

The field axioms imply the following statements, for any $ x,y,z\in F$.

(a) $0x=0$.

(b) If $x\neq 0$ and $y\neq 0$ then $xy\neq 0$.

(c) $(-x)y=-(xy)=x(-y)$.

(d) $(-x)(-y)=xy$.

Proof, Omitted.

1.17 Definition

An ordered field is a field $F$ which is also an ordered set, such that

(a) $x+y<x+z$, if $x,y,z\in F$ and $y<z$.

(b) $xy>0$, if $x\in F, y\in F,x>0,y>0$.

If $x>0(x<0)$, we call $x$ positive(negative).1

1.18 Proposition

The following statements are true in every ordered field.

(a) If $x>0$ then $-x<0$ and vice versa.

(b) If $x>0$ and $y< z$ then $ xy< xz $.

(c) If $x<0$ and $y< z$ then $ xy>xz $.

(d) If $x\neq0$ then $x^2>0$. In particular, $1>0$.

(e) If $0<x<y$ then $0<1/y<1/x$.

Proof, Omitted.

The Complex Field

1.24 Definition

A Complex number is an ordered pair $(a,b)$ of real numbers.

Let $x=(a,b),y=(c,d)$, and $x=y$ if and only if $a=c,b=d$.

Addition is defined as $x+y=(a+c,b+d)$.

Multiplication is defined as $xy=(ac-bd,ad+bc)$.

1.25 Theorem

These definitions of addition and multiplication turn the set of all complex numbers into a field, with $(0,0)$ and $(1,0)$ in the role of $0$ and $1$.

Proof, Verify the field axioms.

Euclidean Spaces

1.36 Definition

For each positive integer $k$, let $\mathbb{R}^k$ be the set of all ordered $k$-tuples, $\mathbf{x}=(x_1,\ldots,x_k)$, where $x_1,\ldots,x_k\in \mathbb{R}$ is called coordinate of $\mathbf{x}$. The elements in $\mathbb{R}^k$ are called points. If $\mathbf{y}=(y_1,\ldots,y_k)$ and if $\alpha$ is a real number, put

so that $\mathbf{x}+\mathbf{y}\in\mathbb{R}^k$ and $\alpha\mathbf{x}\in\mathbb{R}^k$. $\mathbb{R}^k$ is a vector space over the real field. The zero element of $\mathbb{R}^k$ is $\mathbf{0}=(0,\ldots,0)$.

Inner product(scalar product), $\mathbf{x}\cdot\mathbf{y}=\displaystyle\sum_{i=1}^k x_i y_i$.

The norm of $\mathbf{x}$, $|\mathbf{x}|=(\mathbf{x}\cdot\mathbf{x})^{1/2}=\left(\displaystyle\sum_{i=1}^k x_i^2\right)^{1/2}$.

The structure defined is called Euclidean $k$-space.

1.37 Theorem

$\mathbf{x},\mathbf{y},\mathbf{z}\in\mathbb{R}^k$, and $\alpha$ is real. Then,

(a) $|\mathbf{x}|\geqslant 0$.

(b) $|\mathbb{x}=0|=0 \Leftrightarrow \mathbf{x}=\mathbf{0}$.

(c) $|\alpha\mathbf{x}|=|\alpha||\mathbf{x}|$.

(d) $|\mathbf{x}\cdot\mathbf{y}|\leqslant|\mathbf{x}||\mathbf{y}|$.

(e) $|\mathbf{x}+\mathbf{y}|\leqslant|\mathbf{x}|+|\mathbf{y}|$.

(f) $|\mathbf{x}-\mathbf{z}|\leqslant|\mathbf{x}-\mathbf{y}|+|\mathbf{y}-\mathbf{z}|$.

Proof, Omitted.

Appendix

Theorem 1.19 will be proved in this appendix by constructing $\mathbb{R}$ from $\mathbb{Q}$.

We denote rational numbers with English letters $p,q,r,\ldots$,while Greek letters $\alpha,\beta,\gamma,\ldots$, are used to denote the cuts.

This method used here were invented by Dedekind.

(Cantor constructed $\mathbb{R}$ from $\mathbb{Q}$ by means of Cauchy sequences.)

Step 1

The members of $\mathbb{R}$ will be certain subsets of $\mathbb{Q}$, called cuts.

Definition of a cut: any set $\alpha\subset\mathbb{Q}$ with the following properties.

  1. $\alpha$ is not empty, and $\alpha\neq\mathbb{Q}$.
  2. If $p\in\alpha,q\in\mathbb{Q}$, and $q<p$, then $q\in\alpha$.
  3. If $p\in\alpha$, then $p<r$ for some $r\in\alpha$.

Some facts:

  • $\alpha$ has no largest member.
  • If $p\in\alpha, q\notin\alpha$ then $p<q$.
  • If $r\in\alpha, r<s$ then $s\notin\alpha$.

Step 2

Define ‘’$\alpha<\beta$’’ to mean: $\alpha\subsetneq\beta$.

This meets the requirement of Def. 1.5.(Why?)

And only one in three relations can hold for any pair $\alpha,\beta$.

$\mathbb{R}$ is an ordered set.

The ordered set $\mathbb{R}$ has the least-upper-bound property.

Let $A\subset\mathbb{R},A\neq\varnothing$, and assume that $\beta\in\mathbb{R}$ is an upper bound of $A$.

Define $\gamma$ to be the union of all $\alpha\in A$. ( If and only if $ p\in\alpha$ for some $\alpha\in A$)
