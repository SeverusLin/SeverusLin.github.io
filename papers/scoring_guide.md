# AI Relevance Scoring Guide

The AI evaluates how closely a target paper relates to each reference paper using a 0–10 scale:

| Score | Meaning                                      |
|-------|----------------------------------------------|
| 0‑2   | Completely unrelated (different field)       |
| 3‑4   | Weakly related (same broad area, different subfield) |
| 5‑6   | Moderately related (shares some concepts/techniques) |
| 7‑8   | Strongly related (same subfield, similar methods)    |
| 9‑10  | Highly related (very close topic, potential overlap) |

This guide is used by the AI relevance evaluator and is also linked from the daily paper page.
